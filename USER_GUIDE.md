# Deskia Prompt Library - User Guide

## What is Deskia Prompt Library?
A centralized platform for storing, organizing, and sharing AI prompts across your team. Save your best prompts, categorize them, and never lose a great prompt again.

---

## Getting Started

### 1. Create Your Account
- Navigate to the application URL
- Click **Register** and enter your email, password, and full name
- Check your email for verification (if required)
- Log in with your credentials

### 2. Reset Password (if needed)
- Click **Forgot Password** on the login page
- Enter your email address
- Check your email for the reset link

## Core Features

### üìù Managing Prompts

#### Create a New Prompt
1. Click the **"+ New Prompt"** button on your dashboard
2. Fill in the details:
   - **Title**: Short, descriptive name for your prompt
   - **Description**: What the prompt does (optional)
   - **Prompt Content**: Your actual AI prompt text
   - **Category**: Select from existing categories
   - **Tags**: Add searchable keywords (comma-separated)
   - **Visibility**: Toggle "Public" to share with all users
3. Click **Save**

#### Edit a Prompt
1. Find your prompt in the list
2. Click the **Edit** (pencil) icon
3. Make your changes
4. Click **Save**

#### Delete a Prompt
1. Find your prompt in the list
2. Click the **Delete** (trash) icon
3. Confirm deletion

#### Copy a Prompt
- Click the **Copy** icon to copy the prompt content to your clipboard
- Paste it into your AI tool (ChatGPT, Claude, etc.)

#### Track Usage
- Each time you copy a prompt, the usage count increments
- View usage statistics to see your most-used prompts

---

### üîç Finding Prompts

#### Search
- Use the search bar to find prompts by:
  - Title
  - Description
  - Content
  - Tags

#### Filter by Category
- Click on a category in the sidebar to view only prompts in that category
- Categories are color-coded for easy identification

#### Filter by Visibility
- **My Prompts**: View only prompts you created
- **Public Prompts**: View prompts shared by all users
- **All Prompts**: View everything you have access to

## Admin Features (Admin Users Only)

### üë• User Management
1. Navigate to **Admin Panel** from the sidebar
2. Go to **User Management** tab
3. View all registered users
4. Change user roles:
   - **User**: Standard access
   - **Admin**: Full access + admin panel

### üè∑Ô∏è Category Management
1. Navigate to **Admin Panel** from the sidebar
2. Go to **Category Management** tab

#### Create a Category
1. Click **"+ Add Category"**
2. Enter:
   - **Name**: Category name
   - **Description**: What this category contains
   - **Color**: Pick a color for easy identification
3. Click **Create**

#### Edit a Category
1. Click the **Edit** icon next to the category
2. Update details
3. Click **Save**

#### Delete a Category
1. Click the **Delete** icon
2. Confirm deletion
   - **Note**: Prompts in this category won't be deleted, but will become uncategorized

## Best Practices

### ‚úÖ Organizing Prompts
- Use clear, descriptive titles
- Add relevant tags for better searchability
- Categorize prompts logically
- Keep descriptions concise but informative

### ‚úÖ Writing Effective Prompts
- Be specific and clear
- Include context when needed
- Use examples in your prompts
- Test prompts before saving
- Update prompts when you improve them

### ‚úÖ Sharing with Team
- Mark well-tested prompts as **Public**
- Keep experimental prompts **Private**
- Use consistent naming conventions
- Add usage notes in descriptions

### ‚úÖ Maintenance
- Review and update old prompts regularly
- Delete outdated or duplicate prompts
- Consolidate similar prompts
- Monitor usage counts to identify favorites
